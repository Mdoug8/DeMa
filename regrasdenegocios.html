Código Descrição
RDN001 As informações prestadas pelo usuário deverá ser mantido em segurança.
RDN002 Para o cadastro do cartão será necessário o preenchimento dos dados do titular do cartão.
RDN003 Para efetuar uma compra será necessário que a opção de entrega esteja selecionada de forma correta.
RDN004 Para poder efetuar cadastro de produtos é necessário que o vendedor não esteja com os dados incompletos ou incorretos.
RDN005 No recebimento do produto cabe ao vendedor entregar o produto de acordo como foi solicitado.
